## Problem Statement
Small and medium online businesses in Nepal primarily sell through social media messaging platforms. These platforms are not designed for managing orders, inventory, or customer follow-ups, leading to missed inquiries, manual errors, and limited scalability.

## Project Overview
The project focuses on using AI to transform unstructured social media conversations into structured business operations. The system aims to reduce manual work, improve response efficiency, and provide better visibility into orders and business performance.

## Hackathon Scope
- Message understanding from social platforms  
- AI-assisted conversation handling  
- Order detection and tracking  
- Basic analytics and insights
